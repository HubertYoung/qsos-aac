# Kotlin_AAC
## 项目介绍
安卓AAC框架，完全由Kotlin编写、模块化。 
集成RxJava、Retrofit、ARouter、Room、Lifecycle等常用框架。 

## 已有业务
- 图虫

## 待加入业务
- 二维码（2019）
- 视频录制与播放（2019）
- 拍照、相册与文件选择上传（2019）
- 附件（图片、文档、视频）预览（2019）
- 自定义表单（文本输入、单选多选、时间选择、人员选择、附件选择）（2019）

## 架构说明
- library 基础模块
- - base aac架构、依赖配置、通用工具类、页面路由地址管理、实体对象
- - netservice retrofit/okhttp/gson网络请求管理依赖、上传下载管理
- - widget 自定义控件依赖

- module 业务模块
- - common 通用业务处理依赖
- - tuchong 业务-图虫

- app app整体业务整合、混淆配置
